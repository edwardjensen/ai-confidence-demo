Tokenization Demo

Discover how AI models break down text into smaller units called "tokens" - the fundamental building blocks that AI systems use to understand and process language.

  • Real-time text tokenization visualization
  • Interactive token exploration with detailed explanations
  • Token statistics and compression ratios
  • No API key required - runs entirely in your browser

Embeddings Visualization

Explore how AI models represent words and concepts as high-dimensional vectors. This interactive scatterplot shows semantic relationships and clustering patterns in embedding space.

  • Interactive 2D scatterplot of word embeddings
  • Category-based coloring (emotions, animals, actions, etc.)
  • Multiple dimension reduction techniques (t-SNE, PCA, UMAP)
  • Click-to-explore word details and similarities

Confidence Visualization

See how confident AI models are in their word choices. This demo visualizes the probability scores behind each token, revealing the uncertainty and decision-making process of AI.

  • Color-coded confidence levels for each word
  • Interactive tooltips showing probability scores
  • Adjustable temperature settings to control randomness
  • Token usage tracking and response analysis
